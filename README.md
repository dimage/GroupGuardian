# 🛡️ GroupGuardian — Telegram бот-модератор

[![GitHub stars](https://img.shields.io/github/stars/dimage/GroupGuardian)](https://github.com/dimage/GroupGuardian/stargazers)
[![GitHub issues](https://img.shields.io/github/issues/dimage/GroupGuardian)](https://github.com/dimage/GroupGuardian/issues)
[![GitHub license](https://img.shields.io/github/license/dimage/GroupGuardian)](https://github.com/dimage/GroupGuardian/blob/main/LICENSE)

**GroupGuardian** — это открытый, гибкий и масштабируемый бот для модерации групп Telegram. Он автоматизирует рутинные задачи, даёт администраторам точечные инструменты воздействия (например, `/nomedia`, `/shadowban`) и предоставляет прозрачную аналитику.

> 📌 **Статус проекта:** Концепция и документация готовы. Код в активной разработке. Следите за обновлениями!

---

## 🚀 Основные возможности

- **Система предупреждений** — гибкие лимиты, сроки действия, автоматический бан при превышении.
- **Уникальная команда `/nomedia`** — запрет на отправку медиа, стикеров и ссылок для конкретного пользователя (текст разрешён).
- **Теневой бан (`/shadowban`)** — пользователь видит свои сообщения, но другие участники их не видят.
- **Пре-модерация** — все сообщения проходят проверку модераторами перед публикацией.
- **Защита от рейдов** — автоматическое включение режима «тишины» при массовом вступлении.
- **Анти-спам и анти-флуд** — фильтр слов, обнаружение дублей, капча для новых участников.
- **Система ролей** — разделение прав между супер-админами, старшими и младшими модераторами.
- **Кастомные команды** — создавайте свои ответы на ключевые слова (например, `!rules`).
- **Голосования и опросы** — создавайте опросы с защитой от накрутки.
- **Статистика и отчёты** — еженедельные отчёты и команда `/stats`.
- **Админ-панель** — веб-интерфейс для управления данными и просмотра логов.
- **Мониторинг** — интеграция с Prometheus и Grafana (метрики, дашборды).
- **Готов к масштабированию** — использует PostgreSQL, Redis, Docker.

## 🏗 Архитектура

| Компонент | Технология |
|-----------|------------|
| Язык | Python 3.11+ |
| Фреймворк | aiogram 3.x |
| База данных | PostgreSQL |
| Кэш и состояния | Redis |
| Веб-интерфейс | Flask-Admin + AdminLTE |
| Мониторинг | Prometheus + Grafana |
| Контейнеризация | Docker + Docker Compose |

> **Важно:** PostgreSQL, Redis, Grafana и Prometheus уже развёрнуты в отдельных контейнерах и подключаются через переменные окружения.

> **Сейчас код в разработке.** Ожидайте релиз MVP в ближайшее время.

После завершения разработки установка будет выглядеть так:

## 🗺️ План разработки

Разработка ведётся по этапам (подробности в [`06_roadmap_and_recommendations.md`](docs/06_roadmap_and_recommendations.md)):

1. **MVP** — базовая модерация (warn, mute, ban, nomedia).
2. **Расширенная модерация** — система предупреждений, анти-флуд, капча, shadowban.
3. **Администрирование** — роли, кастомные команды, админ-панель.
4. **Продвинутые функции** — пре-модерация, защита от рейдов, голосования.
5. **Аналитика и безопасность** — метрики, отчёты, бэкапы, 2FA.
