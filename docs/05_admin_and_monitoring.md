## 🖥 Админ-панель (Flask-Admin)

### Доступ

- Веб-интерфейс: `http://localhost:5000/admin`
- Защита: авторизация через логин/пароль из `.env`
- Дополнительно: двухфакторная аутентификация (TOTP)

### Разделы админ-панели

| Раздел | Описание | Действия |
|--------|----------|----------|
| **Панель управления** | Общая статистика по всем чатам | Просмотр |
| **Пользователи** | Все пользователи, их данные, активные ограничения | Просмотр, редактирование, поиск |
| **Чаты** | Все чаты, настройки, состояние модерации | Просмотр, редактирование, настройка |
| **Предупреждения** | История всех предупреждений | Просмотр, фильтрация, снятие |
| **Ограничения** | Активные `/nomedia`, `/shadowban` | Просмотр, снятие |
| **Капча** | Статистика прохождения, сессии, нарушения | Просмотр, сброс сессий, ручной пропуск |
| **Запрещённые слова** | Списки слов для каждого чата | Добавление, удаление, редактирование |
| **Роли админов** | Назначенные роли и права | Назначение, изменение |
| **Логи** | Все действия администраторов | Просмотр, фильтрация, экспорт |
| **Жалобы** | Заявки `/report` | Просмотр, обработка, закрытие |
| **Кастомные команды** | Список всех кастомных команд | Добавление, удаление |
| **Экспорт данных** | Выгрузка данных по чатам | `.csv`, `.xlsx`, `.json`, `.yaml` |
| **Статистика** | Графики и диаграммы (Chart.js) | Просмотр, фильтрация по датам |

### Скриншоты (описание)

1. **Дашборд:** карточки с ключевыми метриками (всего пользователей, варнов, банов)
2. **Графики:** динамика нарушений по дням, топ-5 нарушителей
3. **Таблицы:** все данные с пагинацией и фильтрацией

## 📊 Мониторинг (Prometheus + Grafana)

### Метрики для сбора

| Категория | Метрика | Описание |
|-----------|---------|----------|
| Сообщения | `messages_total` | Всего сообщений в чатах |
| Сообщения | `messages_per_chat` | Сообщений по чатам |
| Модерация | `warnings_total` | Всего предупреждений |
| Модерация | `mutes_total` | Всего мутов |
| Модерация | `bans_total` | Всего банов |
| Модерация | `deleted_messages_total` | Всего удалённых сообщений |
| Капча | captcha_passed_total | Всего успешно пройденных капч |
| Капча | captcha_failed_total | Всего неудачных попыток (ошибок) |
| Капча | captcha_timeout_total | Всего таймаутов (не прошли за 60 сек) |
| Капча | captcha_bans_temporary | Временных банов за 1-ю ошибку |
| Капча | captcha_bans_permanent | Вечных банов за 2-ю ошибку |
| Капча | captcha_sessions_active | Текущее количество активных сессий |
| Производительность | `bot_response_time` | Время ответа бота |
| Производительность | `db_query_time` | Время выполнения запросов к БД |
| Производительность | `redis_cache_hits` | Попадания в кэш Redis |
| Активность | `active_users_daily` | Активные пользователи в день |
| Активность | `new_users_daily` | Новые пользователи в день |
| API | `telegram_api_calls` | Количество вызовов Telegram API |
| Ошибки | `errors_total` | Количество ошибок (по типу) |



### Дашборды Grafana

| Дашборд | Назначение |
|---------|------------|
| **Overview** | Общая картина: активность, нарушения, производительность |
| **Moderation** | Детальная статистика по варнам, мутам, банам |
| **Performance** | Время ответа, нагрузка на БД и Redis |
| **User Activity** | Активность пользователей, новые участники |
| **Captcha** | Статистика по капче: прохождения, ошибки, баны, таймауты |

### Настройка

```yaml
# prometheus.yml
scrape_configs:
  - job_name: 'telegram_bot'
    static_configs:
      - targets: ['bot:8000']
```

## 🔒 Безопасность

### 1. Аутентификация в админ-панели

```python
# Двухфакторная аутентификация (TOTP)
import pyotp

totp = pyotp.TOTP('base32secret')
code = totp.now()  # Текущий код
```
### 2. Защита от атак

| Угроза | Защита |
|--------|--------|
| Брутфорс | Rate Limiting (ограничение по IP) |
| SQL-инъекции | SQLAlchemy (ORM с параметризацией) |
| XSS | Экранирование вывода в Flask |
| CSRF | CSRF-токены в формах |
| Подбор пароля | Блокировка после 5 неудачных попыток |

### 3. Бэкапы

```bash
# Ежедневный дамп PostgreSQL (cron)
0 3 * * * pg_dump -U postgres -d telegram_bot > /backups/db_$(date +\%Y\%m\%d).sql

# Автоматическая загрузка в облако (S3/Google Cloud)
aws s3 cp /backups/ s3://my-bucket/backups/ --recursive
```

### 4. Логирование

| Тип лога | Назначение | Хранение |
|----------|------------|----------|
| Логи бота | Отслеживание работы бота | Файлы + Sentry |
| Логи админов | Аудит действий | Таблица `admin_logs` |
| Логи ошибок | Отслеживание исключений | Sentry |


## 📈 Аналитика (Amplitude / PostHog)

### Отслеживаемые события

| Событие | Параметры |
|---------|-----------|
| `user_joined` | `chat_id`, `user_id` |
| `user_left` | `chat_id`, `user_id` |
| `warning_given` | `user_id`, `reason` |
| `warning_expired` | `user_id` |
| `mute_applied` | `user_id`, `duration` |
| `ban_applied` | `user_id`, `reason` |
| `nomedia_applied` | `user_id`, `duration` |
| `message_deleted` | `chat_id`, `reason` |
| `command_used` | `command_name`, `user_role` |
| `captcha_started` | `user_id`, `chat_id`, `task_id` |
| `captcha_passed` | `user_id`, `chat_id`, `attempts` |
| `captcha_failed` | `user_id`, `chat_id`, `attempts` |
| `captcha_timeout` | `user_id`, `chat_id` |
| `captcha_banned_temporary` | `user_id`, `chat_id`, `duration` |
| `captcha_banned_permanent` | `user_id`, `chat_id` |
