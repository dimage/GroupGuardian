## Часть 1: Общая концепция и цели

---

## 🎯 Назначение и философия

**GroupGuardian** — это открытый, гибкий и масштабируемый бот для модерации групп Telegram. Он автоматизирует рутинные задачи, даёт администраторам точечные инструменты воздействия, обеспечивает прозрачность и аналитику, а также предоставляет уникальные функции, отсутствующие в стандартных решениях.

### Ключевые проблемы, которые решает бот:

| Проблема | Решение |
|----------|---------|
| Спам и флуд | Анти-спам фильтры, обнаружение дублей, интерактивная капча при входе (выбор эмодзи) с прогрессивной системой банов |
| Токсичное поведение | Система предупреждений, автоматические наказания |
| Реклама и ссылки | Блокировка ссылок, фильтр слов |
| Нагрузка на админов | Автоматизация 80% рутинных задач |
| Отсутствие прозрачности | Логирование всех действий, админ-панель |
| Массовые атаки (рейды) | Автоматическая защита от рейдов |

### Главные отличия от конкурентов

1. **Уникальная команда `/nomedia`** — точечное ограничение на отправку медиа, стикеров и ссылок (текст разрешён).
2. **Теневой бан (`/shadowban`)** — тихое наказание без провокации конфликтов.
3. **Пре-модерация сообщений** — полный контроль над публикуемым контентом.
4. **Полностью открытый код** — вы владеете своими данными и можете дорабатывать бота под свои нужды.
5. **Профессиональный стек** — PostgreSQL, Redis, Docker, Prometheus, Grafana.
6. **Прогрессивная система капчи** — первая ошибка → временный бан, вторая → вечный, что эффективно борется с рейдерами.

---

## Капча при входе новых участников

При вступлении нового пользователя бот мгновенно ограничивает его права на отправку сообщений и медиа, после чего предлагает пройти проверку: выбрать правильный эмодзи для заданного слова. Это позволяет отсеивать ботов и рейдеров до того, как они смогут навредить чату.

**Сценарии прохождения:**
- **Успех** — права восстанавливаются, пользователь получает доступ к чату.
- **Ошибка (1-й раз)** — временный бан на 10 минут.
- **Ошибка (2-й раз при повторном входе)** — вечный бан.
- **Таймаут (60 секунд)** — кик без бана (можно вернуться позже).

## 🏗 Архитектура и стек технологий

### Основные компоненты

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| Язык | Python 3.11+ | Основной язык разработки |
| Фреймворк | aiogram 3.x | Асинхронная работа с Telegram API |
| База данных | PostgreSQL | Основное хранилище данных |
| Кэш и состояния | Redis | Хранение временных состояний, TTL, кэширование |
| Веб-интерфейс | Flask-Admin + AdminLTE | Админ-панель для управления данными |
| Мониторинг | Prometheus + Grafana | Сбор и визуализация метрик |
| Контейнеризация | Docker + Docker Compose | Упаковка и запуск |
| Аналитика | Amplitude / PostHog | Отслеживание пользовательского поведения |
| Отслеживание ошибок | Sentry | Логирование ошибок |
| Миграции БД | Alembic | Управление схемой БД |
| Валидация | Pydantic V2 | Валидация данных |
| Интернационализация | GNU gettext + Babel | Поддержка нескольких языков |

### Инфраструктурные особенности

**Важно:** PostgreSQL, Redis, Grafana и Prometheus уже развёрнуты в отдельном контейнере и не требуют дополнительной настройки. Бот подключается к ним через переменные окружения.

## 🚀 Бизнес-ценность

1. **Экономия времени:** автоматизация рутинных задач освобождает до 80% времени администраторов.
2. **Снижение конфликтов:** прозрачная система наказаний и теневой бан уменьшают напряжённость.
3. **Улучшение качества контента:** пре-модерация и фильтры гарантируют соблюдение правил.
4. **Масштабируемость:** бот работает в нескольких чатах одновременно с индивидуальными настройками.
5. **Безопасность данных:** вы полностью контролируете свои данные (Open Source + локальное хранение).
