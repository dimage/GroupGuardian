# 🤖 GroupGuardian — концепция Telegram-бота-модератора

## Часть 3: Команды бота

---

## 👥 Команды для всех пользователей

| Команда | Описание | Пример |
|---------|----------|--------|
| `/start` | Приветствие, краткая справка | `/start` |
| `/help` | Подробная справка по командам | `/help` |
| `/rules` | Показать правила чата | `/rules` |
| `/report @user [причина]` | Пожаловаться на пользователя | `/report @spammer Спам` |
| `/mywarns` | Показать количество активных предупреждений | `/mywarns` |
| `/profile` | Показать рейтинг/статистику | `/profile` |
| `!команда` | Выполнить кастомную команду | `!site` |

---

## 🛡️ Команды для администраторов

### Базовые команды модерации

| Команда | Описание | Уровень доступа |
|---------|----------|-----------------|
| `/warn @user [причина]` | Выдать предупреждение | Все админы |
| `/unwarn @user` | Снять одно предупреждение | Модератор+ |
| `/mute @user <время> [причина]` | Заглушить (10m, 1h, 1d) | Модератор+ |
| `/unmute @user` | Снять заглушение | Модератор+ |
| `/kick @user [причина]` | Исключить из чата | Модератор+ |
| `/ban @user [причина]` | Забанить | Модератор+ |
| `/unban @user` | Разбанить | Старший модератор+ |
| `/clean <N>` | Удалить N последних сообщений | Модератор+ |
| `/shadowban @user [время]` | Теневой бан | Старший модератор+ |
| `/unshadowban @user` | Снять теневой бан | Старший модератор+ |

### Команды ограничений

| Команда | Описание | Уровень доступа |
|---------|----------|-----------------|
| `/nomedia @user [время]` | Запретить медиа, стикеры и ссылки | Модератор+ |
| `/unmedia @user` | Снять ограничение | Модератор+ |

### Настройка чата

| Команда | Описание | Уровень доступа |
|---------|----------|-----------------|
| `/setlang <ru/en>` | Язык интерфейса | Супер-админ |
| `/setwelcome <текст>` | Установить приветствие | Супер-админ |
| `/setgoodbye <текст>` | Установить прощание | Супер-админ |
| `/addword <слово>` | Добавить слово в фильтр | Модератор+ |
| `/removeword <слово>` | Удалить слово | Модератор+ |
| `/listwords` | Показать список слов | Модератор+ |
| `/setwarnlimit <число>` | Лимит предупреждений до бана | Супер-админ |
| `/setmutetime <время>` | Время мута по умолчанию | Супер-админ |
| `/setrole @user <role>` | Назначить уровень доступа (owner/admin/moderator) | Супер-админ |
| `/setmoderate on/off` | Включить пре-модерацию | Супер-админ |
| `/setantiraid on/off` | Включить защиту от рейдов | Супер-админ |
| `/setautoclean <N>` | Автоочистка сообщений старше N дней | Супер-админ |

### Настройка капчи
| Команда | Описание | Уровень доступа |
|---------|----------|-----------------|
| `/setcaptcha on/off` | Включить/выключить капчу при входе новых участников | Супер-админ |
| `/setcaptchatimeout <секунды>` | Установить таймаут на прохождение капчи (по умолчанию 60) | Супер-админ |
| `/setcaptchaban <минуты>` | Установить длительность первого бана за ошибку (по умолчанию 10) | Супер-админ |

### Кастомные команды

| Команда | Описание | Уровень доступа |
|---------|----------|-----------------|
| `/addcmd <ключ> <ответ>` | Создать кастомную команду (например, !rules) | Супер-админ |
| `/delcmd <ключ>` | Удалить команду | Супер-админ |
| `/listcmds` | Показать все кастомные команды | Модератор+ |

### Голосования

| Команда | Описание | Уровень доступа |
|---------|----------|-----------------|
| `/poll <вопрос> <варианты>` | Создать опрос | Модератор+ |
| `/closepoll <id>` | Закрыть опрос досрочно | Модератор+ |

### Статистика

| Команда | Описание | Уровень доступа |
|---------|----------|-----------------|
| `/stats` | Показать статистику чата | Модератор+ |
| `/reportstats` | Отправить еженедельный отчёт | Супер-админ |

---

## 🔐 Уровни доступа администраторов

| Уровень | Название | Возможности |
|---------|----------|-------------|
| **owner** | Супер-админ | Все команды, включая назначение ролей |
| **admin** | Старший модератор | Все команды, кроме назначения ролей |
| **moderator** | Младший модератор | Базовые команды (warn, mute, kick, clean, nomedia) |

---

## 📝 Форматы времени

При указании времени в командах используются латинские суффиксы:

| Суффикс | Значение | Пример |
|---------|----------|--------|
| `m` | минуты | `30m` |
| `h` | часы | `1h` |
| `d` | дни | `2d` |
| `w` | недели | `1w` |
